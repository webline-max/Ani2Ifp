# Ani2Ifp
Console tool for converting .ani animation files to .ifp format

ANI to IFP Converter

Simple command-line tool that converts .ani animation files to .ifp format.

Usage

1. Place ani2ifp.exe in the folder with your .ani files
2. Run the executable
3. All .ani files will be converted to .ifp

Building

```bash
cl /EHsc /std:c++17 ani2ifp.cpp
```
